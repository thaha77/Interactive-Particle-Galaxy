# Interactive Particle Galaxy 

A high-performance, interactive visual web project built using HTML5 Canvas and pure JavaScript. The particles dynamically react to mouse movements and touch inputs based on custom mathematical physics, creating a smooth galaxy-like effect.

# Features
- *Fluid Particle Physics:** Particles disperse when approached by the cursor/touch and smoothly return to their original position.
- *Neon Aesthetic:** High-quality glowing particle effects with custom colors.
- *Fully Responsive:** Adapts instantly to all screen sizes (Desktop, Tablet, and Mobile).
- *Zero AI / Pure Logic:** Built entirely using vanilla JavaScript logic, intervals, and mathematical formulas.

## Tech Stack Used
- **Frontend:** HTML5 (Canvas API), CSS3
- **Logic:** Vanilla JavaScript (No external libraries used)
